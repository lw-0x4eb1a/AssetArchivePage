常见问题

软件支持哪些平台？¶

本软件只支持“可以玩饥荒”的设备，如Windows系统和MacOS系统。

以下平台不会提供支持：

  • 网页版/手机端 - 这些平台无法安装饥荒，或无法读取本地的游戏文件。
  • Windows 7 及更早的系统 - Steam已停止对这些系统的支持。
  • Windows 32位 - 软件架构原因。
  • MacOS 10.12 及更早的系统 - 软件架构原因。
  • Linux系统 - 此平台通常用于托管饥荒服务器，而不是直接玩游戏。

是否支持饥荒单机版？¶

目前不支持，请等待后续更新。
